# 🎓 Student Management System

## 📌 Description
A desktop application to manage student records using Python and MySQL with a simple GUI.

---

## 🚀 Features
- Add Student
- Update Student
- Delete Student
- Search Students
- Login System

---

## 🛠️ Tech Stack
- Python
- MySQL
- Tkinter / CustomTkinter
- Pandas

---

## ▶️ How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run:
python main.py

---

## 📂 Project Structure
- main.py
- gui.py
- db.py
- login.py
